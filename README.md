# Build Beyond — Company Website

Marketing/services website, a software service company offering custom software, platforms, LMS, and dedicated development teams.

## Tech Stack

- **Vite + React + TypeScript**
- **Tailwind CSS** with shadcn-ui components
- **Framer Motion** for scroll animations
- **Vercel** for hosting + serverless functions
- **Brevo (Sendinblue)** SMTP for contact form emails

## Commands

```sh
npm run dev          # Start dev server (port 8080)
npm run build        # Production build
npm run lint         # ESLint check
npm run preview      # Preview production build locally
npm run test         # Run tests
```

## Deployment

Deployed via Vercel. Push to `main` to trigger a production deploy.
